## 命令组成



```
[数字] + [operator] + [数字] + [motion]
```

光标移动的方式：motion

| motion       | 含义                                   |
| ------------ | -------------------------------------- |
| `h, j, k, l` | 左下上右                               |
| `w, b, e`    | 后面的词\前面的词\移光标至词末         |
| `$`          | 移光标至行末                           |
| `0`          | 移光标至行首                           |
| `^`          | 移光标至本行首个非空格的字节           |
| `gg`         | 移光标至整个文本的首行首个非空格字节   |
| `G`          | 移光标至整个文本最后一行首个非空格字节 |
| `%`          | 移光标至匹配括号的另一端               |
| `J`          | 合并行                                 |

以行为单位时：

```
[数字] + [operator] + [数字] + [operator]
```

操作命令：operator

| 命令         |  含义  |
| ------------ | ---- |
| `d` |     删除 |
| `c` |     替换 |
| `<, >` |  左缩进、右缩进  |
| `y` |   复制   |
| `gu, gU` | 变换为小写、大写 |
| `~` | 大小写转换 |
